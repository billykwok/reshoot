# Installation

As mentioned in [Overview](/?path=/docs/documentation-overview--page#what-is-in-the-box), it is not mandatory to use all 3 available packages, despite using them in combination yields the best result. Assuming all are used, you can install them using the following Yarn commands.

```sh
# yarn
yarn add -D @reshoot/loader @reshoot/macro
yarn add @reshoot/react

# If babel-plugin-macros is not installed
yarn add -D babel-plugin-macros
```

Or, if you use npm,

```sh
# npm
npm install -D @reshoot/loader @reshoot/macro
npm install @reshoot/react

# If babel-plugin-macros is not installed
npm install -D babel-plugin-macros
```
