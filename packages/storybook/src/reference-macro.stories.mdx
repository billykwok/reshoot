# @reshoot/macro

## Configuration

As a syntactic sugar for `@reshoot/loader`, `@reshoot/macro` itself is not configurable. All options passed to it are transpiled into query parameters in `require` call. Thus, `@reshoot/macro` accepts the same set of options as `@reshoot/loader`.

```typescript
import reshoot from '@reshoot/macro';

const image = reshoot('./images/example.jpg', {
  /* options */
});
```

## Available Options

Please refer to the [API reference](?path=/docs/documentation-api-reference-reshoot／loader--page) of `@reshoot/loader` for details.

> The option specified in `@reshoot/macro` is only applied to the corresponding image.
